******** Two Pointer Pattern ********

**** The Trigger ****
- A sorted Array
- Find a pair/tiplet that sums to
- Remove duplicates, reverse, palindrome
- You need to compare elements from both ends or at different speeds]

***** Core Idea *****
Instead of nested loops (O(n²)), use two pointers moving toward each other (or in the same direction) to shrink the search space → O(n).
